Выбираем облачный сервис по стоимости и производительности

Microsoft Azure, Google Cloud и AWS

20 Мая, 2017

Несмотря на то, что с точки зрения маркетинга и "набора фич" облачные сервисы отличаются практически никак, выбор облачного сервиса по одному лишь описанию производителя непростая задача. Сервисы сильно отличаются по архитектуре, а следовательно по перфомансу.

Сервис и размер CPU RAM, GiB Стоимость в час Стоимость в месяц
AWS m4.xlarge 4* 15 $ 0.238 $ 174.22
Azure DS3v2 promo 4 14 $ 0.222 $ 162.06
Google n1-standard-4 4* 16 $ 0.146 $ 106.90

* vCPU

Стоимость взята для европейского региона, операционна система Linux.

Методика тестирования

В Magneta.ai мы работаем с WebSocket используя Python / Tornado. Поэтому я буду оценивать производиельность сервера с помощью простого WebSocket приложения. Репозиторий с тестом https://github.com/pachay/loadtest.

Результаты

Сервис и размер Очки Стоимость месяца за одно очко
AWS m4.xlarge 1177 $ 0.133
Azure DS3v2 promo 1417 $ 0.114
Google n1-standard-4 1243 $ 0.086

Стоимость месяца за одно очко = Стоимость месяца / Количество очков.

За счет честных CPU Azure получила больше очков. Виртульные машины от Microsoft быстрее справляются с WebSocket тестом.

Несмотря на то, что Google Cloud медленнее, стоимость одного очка перфоманса у Google меньше. Соответственно, развернув чат сервер в Google Cloud, мы заплатим меньше денег при той же нагрузке.
